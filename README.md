# Pete Thinkful Portfolio

This project is a personal portfolio website for Pete Thinkful, an artist based in Denver, Colorado. The site showcases background information, selected artwork, and social links for contacting or following Pete.

The project was built as a capstone exercise to demonstrate proper HTML structure, semantic markup, and CSS styling.
